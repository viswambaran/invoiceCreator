Invoice Creator
===============

Installation

1. Extract the ZIP.
2. Double-click "Install Invoice Creator.bat".
3. Wait for installation to finish.

The installer will:

• Install or update Invoice Creator
• Create a Desktop shortcut
• Create a Start Menu shortcut
• Launch the application

Future Updates

Simply download the latest release and run
"Install Invoice Creator.bat" again.

Requirements

Windows
Python 3.10 or newer
